Soil Feeder + Telegram Bot

#IoT #Automation #LoRa #ESP32 #Blynk #Telegram
Soil Feeder Bot Telegram
Master Master-Lora Lora

This project is a fully automated system for filling soil into polybags, designed to reduce manual labor and increase consistency in agriculture tasks. It is built using 3 units of ESP32 that work together over LoRa communication to collect, control and transmit real-time data.

The system features an ultrasonic sensor to detect soil height in each polybag, a weight sensor to ensure proper soil amount, and a liquid crystal display for status visualization. Users interact with the system using a keypad, and a servo motor handles the soil dispensing gate.

Two DC motors manage the soil transport mechanism, powered through an AC to DC converter for efficient power supply. Critical system details such as weight, fill status, and alerts are transmitted via LoRa to a remote ESP32 unit.

For monitoring and control, the system is integrated with Blynk IoT for mobile dashboard feedback and control, and Telegram Bot for instant alert messaging.

This project challenged me to work with multiple communication protocols, sensor fusion, and real-time system control, while implementing energy-efficient and user-friendly IoT features.
